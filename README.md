# test-insta-api

Instagram Graph APIを叩いて、データを可視化するWEBアプリを作成する。  

```
$ docker -v
Docker version 20.10.3, build 48d30b5
```

```
$ docker-compose -v
docker-compose version 1.28.2, build 67630359
```

# コマンド

```
// バックグラウンドで実行
$ docker-compose up -d
```

```
// 停止
$ docker-compose down
```

```
// 完全に削除
$ docker-compose down --rmi all --volumes --remove-orphans
```
